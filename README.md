# matrix-RPNcalc

irbで利用できます．RPN電卓なので，行列を入力後に演算子を入力します．

例
```
enter [[1,2,3],[4,5,6]]
enter [[1,2,3],[4,5,6]]
add
=> [[2,4,6],[8,10,12]]
```

#### 行列の入力
```
enter [[1,2,3],[4,5,6]]
```

#### 和
```
add
```


#### 差
```
sub
```

#### 積
```
mul
```

#### 転置
```
trans
```

#### 行列式
```
det
```

#### 簡約化
```
simple
```
